任意框架构建的Agent，只需几行代码即可在AgentCore Runtime中运行

1️⃣ Agent代码改动 - 仅需4行

关键点: 使用 @app.entrypoint 装饰器包装你的agent函数 
from bedrock_agentcore.runtime import BedrockAgentCoreApp
from strands import Agent

app = BedrockAgentCoreApp()

@app.entrypoint
async def invoke(payload, context=None):
    user_input = payload.get("prompt", "")
    auth_header = context.request_headers.get('Authorization', '')
    
    # 创建你的agent
    agent = Agent(
        model=model,
        tools=tools,
        system_prompt=SYSTEM_PROMPT,
    )
    
    # 调用agent
    response = agent(user_input)
    
    # 返回结果

    }

if __name__ == "__main__":
    app.run()

2️⃣ 配置和部署 - AWS SDK调用

关键点: 使用AgentCore Starter Toolkit进行配置和部署 
from bedrock_agentcore_starter_toolkit import Runtime

agentcore_runtime = Runtime()

# 配置Runtime
agentcore_runtime.configure(
    entrypoint="lab_helpers/lab4_runtime_streaming.py",
    execution_role=execution_role_arn,
    auto_create_ecr=True,
    requirements_file="requirements.txt",
    region=region,
    agent_name="customer_support_agent",
    authorizer_configuration={
        "customJWTAuthorizer": {
            "allowedClients": [client_id],
            "discoveryUrl": discovery_url,
        }
    },
)

# 部署到生产环境
launch_result = agentcore_runtime.launch(
    env_vars={"MEMORY_ID": memory_id},
    auto_update_on_conflict=True
)

print(f"✅ Runtime deployed: {launch_result.agent_arn}")

✨ 优势

框架无关 自动扩展 完整监控 安全认证

• 框架无关: 支持任何Python AI框架 (Strands, LangChain, CrewAI等)
• 自动扩展: 无需管理服务器,自动处理流量波动
• 完整监控: 集成CloudWatch Observability
• 安全认证: 内置JWT认证和权限控制
